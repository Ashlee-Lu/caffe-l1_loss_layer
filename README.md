# L1 Loss Layer in Caffe

This is a implementation of L1 Loss Layer in Caffe.

## Usage

1. Put the files in corresponding location.
2. Compile and test
```cpp
make -j
make test -j
make runtest GTEST_FILTER='L1LossLayerTest/*'
```
